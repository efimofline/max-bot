# Echo-бот для MAX

Мой первый бот для мессенджера MAX на Python. Повторяет присланный текст.

## Что нужно

- Python 3.10+
- Аккаунт в приложении MAX

## Шаг 1. Получить токен

1. Открой приложение MAX.
2. Найди системного бота **@MasterBot**.
3. Напиши ему и следуй инструкциям: создай нового бота → получи **токен** (длинная строка).

## Шаг 2. Установить зависимости

```powershell
pip install -r requirements.txt
```

## Шаг 3. Указать токен

Самый простой способ — переменная окружения (PowerShell):

```powershell
$env:MAX_BOT_TOKEN = "сюда_твой_токен"
```

Либо впиши токен прямо в `bot.py` в строку `TOKEN = ...`.

## Шаг 4. Запустить

```powershell
python bot.py
```

Теперь напиши боту в MAX — он ответит тем же текстом.

## Как это работает

- `Bot(TOKEN)` — подключение к API MAX по твоему токену.
- `Dispatcher` — маршрутизатор: решает, какой обработчик вызвать.
- `@dp.message_created(...)` — обработчик нового сообщения.
- `event.message.answer(...)` — отправка ответа.
- `dp.start_polling(bot)` — бот постоянно спрашивает сервер «нет ли новых сообщений?» (long polling).
